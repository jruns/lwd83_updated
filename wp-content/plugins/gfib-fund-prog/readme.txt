 Block for displaying fundraising progress.

== Description ==

This plugin activates a new WordPress Block to display a horizontal fundraising progress bar. The block accepts a Donations count, a progress count (USD), and a goal count (USD).

The main bar color for progress and the caption text are also configurable.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/gfib-fund-prog` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
